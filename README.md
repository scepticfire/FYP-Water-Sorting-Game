# Introduction


## <u>Concepts taught:</u>
Basic Queue Operations (Dequeue, Enqueue) &
Stack Operations (Pop, Push)

## <u>Objectives & how to play</u>
- There are 3 stacks <b>(LIFO)</b> and 3 queues <b>(FIFO)</b>
![alt text](<images/starting screen.png>)
- Click on stacks/queues which contains a color block and move them around using Operations Buttons
(in this example, a stack is clicked and the <b>pop</b> button appears)
![alt text](<images/pop example.png>)
- Afterwards click on a queue to enqueue the colored block
![alt text](images/enqueued.png)
- The goal is to have queues/stacks containing all of the specific colored blocks
![alt text](<images/victory screen.png>)
